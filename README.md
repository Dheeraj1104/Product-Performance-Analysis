# Product Performance & Profitability Analysis

## Overview
This project analyzes product-wise sales performance using MySQL and Power BI
to identify profitable and underperforming products.

## Tools Used
- MySQL
- Power BI

## Key Analysis
- Revenue and profit by product
- Profit margin calculation
- Product performance classification
- Regional analysis

## Outcome
The dashboard helps businesses focus on high-profit products instead of only
high-selling products.
